## Web Search Routing

**IMPORTANT**: When you need to search the web, DO NOT use the built-in `WebSearch` tool. Instead:

- Use the `Task` tool with `subagent_type='gemini-search'` for web searches
- Or use the `/search [query]` slash command

The gemini-search agent provides:

- Cached results (1hr TTL) for token savings
- Usage analytics via `/search-stats`
- Google web search via Gemini CLI (grounded results)
- Cache clearing via `/clear-cache`
